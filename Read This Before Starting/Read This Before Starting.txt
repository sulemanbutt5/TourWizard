Kindly Follow these steps before Starting:

* You can check this prototype (deliverable 3) without any installation at https://tourwiz.netlify.app

If you want to run the prototype on your PC then follow the following instructions:
1- Download and Install the NodeJS without chocolatey from -> https://nodejs.org/en/download/ .
2- Open the folder named "tourwizard" in command prompt.
3- Write 'npm install' and press Enter in command prompt.
4- After a few minutes the libraries will be installed.
5- Write 'npm start' and press enter in the Command prompt of "tourwizard" folder.
6- The prototype (deliverable 3) pages will be opened in your default browser.
